Centro de monitoreo

Este proyecto fue desarrollado para lograr para monitorear el comportamiento todas las clínicas de salud digna a nivel nacional, el objetivo es tener un control sobre los tiempos de atención, respuesta y las alertas que se desencadenan al tener ciertos números en cada indicador, se realizó en 2 temas, debido a la situación en la que se utiliza este sistema de desarrollaron 2 temas, oscuro y blanco, oscuro para ambientes cerrados y blanco para habitaciones con más luz, debido a que muchos de estos tableros fueron incorporados a los CNR de salud digna donde el área es iluminada y hay una línea de producción constante, la resolucion esta diseñada exclusivamente para resoluciones grandes (4k) debido a su natualeza de constante visualización de datos.
